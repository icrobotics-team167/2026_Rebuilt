# 2026 Rebuilt

Useful gradle commands:

```
# Compile the code
$ ./gradlew build

# Run simulation
$ ./gradlew simulateJava

# Deploy code to robot
$ ./gradlew deploy

# Run replay - have log open in AdvantageScope
$ ./gradlew simulateJava -P replay

# Apply autoformatter (will also check syntax)
$ ./gradlew spotlessApply
```
